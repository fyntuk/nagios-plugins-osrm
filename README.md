nagios-plugins-osrm
===================

Nagios plugins for osrm

Use check_http to check all functions in osrm api

* check_osrm_hello

* check_osrm_locate

* check_osrm_nearest

* check_osrm_viaroute

Requirements
============

We use check_json from https://github.com/c-kr/check_json you must
install it to use these plugins
